# Camel Drawing App

Camel Drawing App is an online, interactive drawing tool built with **HTML, CSS, and JavaScript**. It allows you to create artwork using a variety of tools, including brush, eraser, text, and shapes (planned). The app is designed to be simple to use, while still giving you control over brushes, colors, fonts, and text formatting.

---

## Features (Current)
- **Brush**: Draw freely with a customizable color and size.
- **Eraser**: Erase parts of your drawing.
- **Text Tool**: Add text to your canvas with selectable font, size, color, bold, italic, and underline.
- **Toolbar**: Only displays controls relevant to the selected tool.
- **Canvas**: 1024x768 default size, ready for expansion with layers and shapes.

---

## How to Use
1. Open the app in a browser.
2. Select a tool from the toolbar.
3. **Brush/Eraser**:
   - Choose color and size.
   - Click and drag on the canvas to draw.
4. **Text Tool**:
   - Click on the canvas to enter text.
   - Select font, size, color, and B/I/U formatting from the toolbar.
5. Toolbar controls appear dynamically depending on the selected tool.
